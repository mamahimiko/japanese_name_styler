# oNAMaE Styler

A playful single-page application built with Next.js, TypeScript, and MUI.

Users can enter their name in different languages, choose a source language, and generate stylish Japanese versions of their name using the DeepL API. The app displays the translated name in both Katakana and Hiragana with multiple Japanese font previews.

The title's "Onamae" means "Name" in Japanese! 



## Features

* Translate names into Japanese using the DeepL API
* Display both Katakana and Hiragana versions
* Preview names in multiple Japanese Google Fonts
* Responsive design for mobile and desktop
* Built with TypeScript and MUI
* Uses React state management and async API handling



## Tech Stack

* Next.js
* React
* TypeScript
* Material UI (MUI)
* DeepL API
* Google Fonts
